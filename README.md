# Creators Vamo!

Site do programa de creators da Vamo!, pronto para publicação no GitHub/Cloudflare Pages.

## Rastreamento Meta

- Pixel exclusivo: **Vamo – Creators**
- Pixel ID: `1558745279374123`
- `PageView`: disparado ao carregar a página
- `Lead`: disparado somente após a confirmação de envio do formulário Tally `5BgVN6`

O evento `Lead` é detectado pelo evento oficial `Tally.FormSubmitted` emitido pelo formulário incorporado.
